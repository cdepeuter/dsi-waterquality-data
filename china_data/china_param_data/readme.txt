China weekly water parameter data from 2016 to now

Each year contains 5 csv file.
Column: weekID
Row: Observation stationID (148 all)

pH.csv: pH* parameter data
DO.csv: DO parameter data, mg/L
COD.csv: COD(Mn) parameter data, mg/L
NH3.csv: NH(3)-N parameter data, mg/L
week_eval: weekly evaluation data.
		  I: clean
		  II: good
		  III: normal
		  IV: bad
		  V: terrible
